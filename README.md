
# Srikanth Madabhushi — AI + ServiceNow Portfolio (GitHub Pages)

This is a starter portfolio to publish on **GitHub Pages**.

## 🚀 Quick Deploy (Web UI — no coding)

1) Create a new repository named **`SrikanthMadabhushi.github.io`** (must match your GitHub username).
2) Upload all files from this folder (`index.html`, `styles.css`, `projects.json`, and the resume file).
3) Go to **Settings → Pages → Build and deployment**:
   - Source: **Deploy from a branch**
   - Branch: **main** (root)
4) Wait 1–2 minutes and visit: `https://SrikanthMadabhushi.github.io`

## 🧱 Update Projects

- Edit `projects.json` to add/update items. Each entry supports:
```json
{
  "module": "ITSM",
  "title": "Agentic AI Auto-Resolution",
  "summary": "Self-healing incident flow ...",
  "ai_focus": "Predictive Intelligence, Agentic AI",
  "integrations": "Jira, Snowflake, Slack/Teams",
  "repo": "https://github.com/..."
}
```
- No build step; the grid renders dynamically on page load.

## 📝 Customize

- Replace the resume file `Srikanth_Madabhushi_AI_ServiceNow_Resume.docx` with your latest version.
- Update your About text in `index.html` (section `<section id="about">`).

## 🛠️ Local Preview (optional)

If you have Python installed, you can preview locally:
```bash
python -m http.server 8080
# then open http://localhost:8080
```
