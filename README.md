
# Srikanth Madabhushi — AI + ServiceNow Portfolio (with Images)

This package adds:
- A hero banner
- A gallery section
- Project card images via `projects.json`

## Deploy on GitHub Pages

1) Create repo `SrikanthMadabhushi.github.io` (public).
2) Upload all files and the `assets` folder.
3) Settings → Pages → Source: Deploy from a branch, Branch: main, Folder: / (root).
4) Open `https://SrikanthMadabhushi.github.io`.

## Update Images
- Replace files in `/assets` with your own (from Unsplash/Pexels, etc.).
- Update `projects.json` `"image"` paths to point to your images.
